# 6 成人 L2 习得中的输入加工

*Bill VanPatten*

想象一位以西班牙语为母语的人在学英语。在一次对话或讨论中，她听到有人说：「The police officer was killed by the robber.」对母语者而言，很清楚死的是警察；但英语学习者可能把这句话理解成「The police officer killed the robber.」为什么学习者会产生这种误解？这不可能是 L1 影响造成的，因为西班牙语里有完全相同的结构：*El policía fue matado por el ladrón.*

再想象另一位以英语为母语的人在正式教学情境中学习西班牙语。她学习了西班牙语的简单过去时（preterit）。一个月后，她听到有人说：*Juan estudió en Cuernavaca*「John studied in Cuernavaca.」然而，她把这句话理解为 John 正在 Cuernavaca 学习，尽管她已经学习并练习过西班牙语过去时的构成，也尽管英语同样清楚地区分过去与现在（例如 studies 与 studied）。为什么她会产生这种误解？

输入加工（input processing，IP）关注的正是这类情形，因为习得在某种程度上是理解的副产品（见例如 Truscott & Sharwood Smith，2004）。尽管理解并不保证习得，但如果没有理解，习得也不可能发生。为什么？因为相当一部分习得依赖学习者在理解过程中建立恰当的**形式—意义联结**。学习者需要输入中的原始数据来建构一个语言系统。相当一部分习得依赖学习者对句子意义的正确解释（Carroll，2001；VanPatten，2004a；VanPatten & Rothman，2014；White，1987）。

本章讨论 IP 的基础内容以及与之相关的研究。你将会看到，IP 并不是一个全面的语言习得理论或模型。相反，它旨在成为一个关于理解过程中发生了什么的理论或模型，而这些过程随后可能影响或与其他过程相互作用。我将从对该理论及其构念的概述开始。

### 理论及其构念

IP 关注三个基本问题，它们都基于这样一个假设：语言习得的一个关键组成部分，是在理解过程中建立形式—意义联结：

- 学习者在什么条件下建立**最初**的形式—意义联结？
- 为什么在某一特定时刻，他们会建立某些形式—意义联结，而不会建立另一些？
- 学习者在理解句子时使用哪些内部心理语言学策略？这些策略可能如何影响习得？

我们用本章引言中的例子来具体说明。在把英语作为 L2 学习时，学习者必须在某个时点把「过去性」（PASTNESS）的意义映射到动词屈折 /-t/（书面形式为「-ed」）上。这是如何发生的？以及，为什么学习者从第一次在明确指涉过去的语境里遇到这一形式时，并不会就这么做？就此而言，IP 是一个关于理解过程中逐毫秒展开的句子加工的模型：在某一时刻，学习者如何把特定形式与特定意义联结起来，或为什么没有联结起来。它也是一个关于学习者如何从输入中提取初始数据来构建语言系统的模型；也就是说，这些数据会被交付给其他加工器与机制，由它们负责真正存储并组织这些数据（例如 UG；见第 3 章[1]）。这一点可如图 6.1所示。

IP 对在理解过程中引导学习者加工输入中语言数据的因素提出了若干主张。这些主张可概括如下。

- 学习者在理解时被驱动去获取意义。
- 对学习者而言，理解在起步阶段在认知加工与工作记忆方面相当费力。不同于 L1 母语者，L2 学习者必须发展出理解能力；在一段时间里，当学习者对句子进行逐毫秒分析时，理解可能会消耗大量计算资源。这会影响 IP 机制会关注什么。
- 同时，学习者的加工容量有限，无法在逐时加工过程中像母语者那样处理并存储同等数量的信息。
- 学习者可能会使用某些 IP 的普遍机制，但也可能使用 L1 的输入加工器（或句法解析器 parser，我们稍后会定义）。

第一个主张引出了 IP 中的一个原则：学习者会通过寻找词汇项来把握意义，尽管其具体实现方式仍不清楚。[2]换言之，学习者进入 L2 习得任务时就知道：语言由词构成。因此，他们最先被驱动去建立具有词汇性质的形式—意义联结。例如，如果他们听到「The cat is sleeping」，并且这句话出现在一只猫确实正在睡觉的语境里，学习者会试图分离出编码 CAT 与 SLEEP 等意义的词汇形式，因为学习者（a）基于以往人类经验，在心智/大脑某处存储着这些概念；（b）知道这些概念很可能对应着某些词，而这些词必然存在于语流之中。更进一步，学习者知道内容词汇项（例如 cat、sleep）与非内容词汇项（例如 the、is）不同，并会先寻找内容词汇项，把它们作为解释句子的积木。因此，在「The cat is sleeping」中，学习者最初可能只建立 cat-CAT 与 sleep-SLEEP 这两种联结（再次提醒读者：此类句子是在有一只猫正在睡觉的语境里说出的）。这些主张被编纂为如下 IP 原则：

> **内容词优先原则。** 学习者会先加工输入中的内容词，然后才加工其他成分。

在这一阶段，学习者很可能并不会加工非内容词，或名词与动词上的屈折；这里的 **process** 特指「真正把意义与形式连接起来」，而非仅仅「注意到」。如果学习者确实加工了非内容词和/或屈折，那么负责数据存储与语法建构的（其他）加工器很可能还无法利用这些信息，便会将其丢弃，从而阻断进一步加工。英语中的助动词 *do* 就是一个例子。学习者起初可能会感知到这个助动词（它几乎总是出现在 *yes/no* 问句的句首，例如 *Do you like Mexican food?*；见后文对「句子位置原则」的定义），但由于在加工早期学习者无法把任何意义附着到它上，*do* 在被初次感知后并不会被进一步加工。

然而，IP 模型对屈折与语法标记等现象还有另一项主张：如果某个标记是冗余的，它可能不会被加工，因为学习者首先专注于获取内容词。加工内容词（也就是内容词优先）会使加工语法标记变得不必要，如果它编码的是同一意义的话。在这种情形中，如果给学习者一个句子，例如「I called my mother yesterday」，他们不会加工时态标记；相反，他们会通过加工时间副词（例如 yesterday、tomorrow）来推断时态。因此，当语法手段存在时，内容词优先原则会影响学习者从输入中提取什么。这被编纂为如下原则：

> **词汇偏好原则。** 当词汇项与语法形式都编码同一语义（「现实世界」）信息时，学习者会先为意义加工词汇项，再加工语法形式。

学习者首先倾向于把语义概念与输入中的内容词汇项联结起来，随后才把编码同一语义概念的语法形式联结起来。这个原则可能带来两种后果。第一种后果是：只有当学习者已经加工并把对应的词汇形式纳入其发展中的语言系统之后，他们才会开始加工冗余的语法标记。因此，过去时标记只有在学习者已经加工并纳入 yesterday、last night 等词汇形式之后，才会被加工与纳入。[3]若如此，词汇偏好原则可以被修订为如下表述：

> **（修订版）词汇偏好原则。** 如果语法形式表达的意义也可以由词汇形式编码（也就是说，该语法标记是冗余的），那么学习者起初不会加工这些语法形式，直到他们拥有可以与之匹配的词汇形式为止。

另一种可能的后果是：学习者可能开始完全依赖词汇形式来获取所有信息，并且永远不会加工输入中的语法标记。在这种情形下，当存在冗余时（也就是说，词汇形式与语法形式表达同一意义，如 PASTNESS/-ed、FUTURE/will、THIRD-PERSON SINGULAR/-s[4]等），词汇项的加工会「覆盖」对语法标记的任何加工需要。无论是哪一种情形，IP 模型的一个预测是：从学习者视角看，词汇项能最大化意义提取，因此学习者会持续把注意力放在加工词汇项上，从而损害语法标记的加工；语法标记会在更晚阶段才被加工。VanPatten 与 Keating（2007）在一项研究中展示了这一点：以英语为 L1 的西班牙语 L2 学习者加工一些西班牙语句子，其中时间副词与动词的时态匹配或不匹配（例如「Yesterday I am talking to John」 vs. 「Now I’m talking to John」）。我们使用眼动追踪（关于眼动追踪的讨论见「什么算作证据？」一节）发现，母语者会停留或「回视」（regress）到动词形式以验证时间指涉；而初级与中级阶段学习者则倾向于停留或回视到副词以验证时间指涉。然而，高级学习者的模式与母语者一致，这表明学习者最终会开始在输入中关注语法屈折以获取时间信息（另见 Ellis & Sagarra，2010；Lee，Cadierno，Glass，& VanPatten，1995）。与此同时，VanPatten 与 Keating 还发现：以西班牙语为 L1 的英语 L2 学习者并不会以西班牙语中偏好动词的方式开始加工英语句子；相反，他们的早期英语学习者呈现出与以英语为 L1 的西班牙语学习者相同的模式——用副词成分来加工句子中的时间指涉——这表明该策略是普遍的，而非依赖 L1 经验。

并非所有语法标记都是冗余的。在英语中，*-ing* 是唯一标记「事件正在进行」（IN PROGRESS）这一语义概念的形式，如「The cat is sleeping [IN PROGRESS]」。在带 *-ing* 的句子里，没有任何词汇层面的 IN PROGRESS 指示。对比之下，「The cat sleeps ten hours everyday」中，「sleeps」上的 -*s* 所对应的意义 [THIRD PERSON, SINGULAR, ITERATIVE] 在词汇层面由「the cat [THIRD-PERSON, SINGULAR]」与「everyday [ITERATIVE]」编码。由于学习者总在寻找意义被编码的方式，如果某个语义概念没有被词汇编码，他们才会转向语法标记，看看该语义概念是否在那里表达。因此，当学习者面对动词形式上的 -*ing* 时，他们会比面对第三人称 -*s* 更早被迫建立这种形式—意义联结，因为后者是冗余的，而前者不是。这引出 IP 的另一个原则：

> **非冗余偏好原则。** 学习者更可能先加工非冗余且有意义的语法标记，再加工冗余且有意义的语法标记。[5]

到目前为止，我们只讨论了承载意义的语法标记，例如名词末尾的 -*s* 表示「多于一个」，*-ing* 表示「正在进行」。但也有一些语法标记（尽管不多）并不承载意义。考虑句子「John thinks that Mary is smart」中的 that。that 编码了什么现实世界的语义信息？它不是时态标记；不是事件是否正在进行或是否反复发生的指示；不是复数标记，也不是其他任何与语义相连的语法装置。你无法像「that’s a cow」或「that’s love」那样，指着世界上的某物并说「that’s a ‘that’」。当然，它有语法功能——把两个句子连接起来（即引入一个嵌套从句）——但它不编码任何语义信息。西班牙语中的形容词一致也是类似情形。在 *el libro blanco*（「the white book」）与 *la casa blanca*（「the white house」）中，并不存在语义原因使得前者必须用 *blanco*、后者必须用 *blanca*。[6]西班牙语只是要求形容词与名词一致。IP 模型认为：这类语言形式特征在输入中的加工会晚于那些能够建立真正形式—意义联结的特征。对应原则是：

> **有意义先于无意义原则。** 学习者更可能先加工有意义的语法标记，再加工无意义的语法标记。

不过，IP 不仅仅关心形式—意义联结。当一个人听到一句话时，无论是 L1 还是 L2，他也会以逐微秒的速度对该句子的句法结构进行计算。这称为**句法解析**（parsing）。例如，在英语中，当一个人听到「The cat …」时，解析机制（称为 *parser*）会做如下处理：the cat = DP（determiner phrase）= subject。这称为 *projection*，因为解析器在投射一个句法结构（也就是说，解析器在对词与词之间的语法关系作出最佳猜测）。如果后面跟着一个动词，解析器可能会沿着这条路径继续。例如，「The cat chased …」：the cat = DP = subject，chased = verb［到目前为止，这个句法投射看起来没问题］。如果接下来出现像「the mouse」这样的短语，解析器可能继续：the cat = DP = subject，chased = verb，the mouse = DP = object；解析完成，句法投射成功，句子被计算并理解。但如果接下来的不是「the mouse」，而是「by the boy」，解析器就必须当场重新分析并对句法结构投射不同结果：the cat = DP = subject，chased = verb，by the boy = 哎呀，不是宾语，因此「the cat chased by the boy」= DP = subject。如果后面跟着一个动词，如「howled」，解析器继续：the cat chased by the boy = DP = subject，howled = verb，解析完成且成功。

上述对解析的描述当然大大简化了。[7]但就本章讨论而言，它足以让我们提出这样一个问题：当学习者在 L2 中解析句子时，如果他们不像处理 L1 句子那样拥有一个充分发展的解析器，他们会如何解析？（我再次忽略了学习者如何在解析过程中感知词边界并分离词语。）第一条路径是：学习者拥有普遍的解析策略（或程序），并在开始接触 L2 输入时应用这些策略。另一条路径是：学习者在与 L2 输入互动时迁移或试图迁移其 L1 的解析策略（或程序）。当我们考察如下英语与西班牙语句子时，这两种立场会变得清晰：

- (1)
  1. Mary hates John.
  2. María detesta a Juan.
  3. A Juan María lo detesta.
- (2)
  1. Mary hates him.
  2. María lo detesta.
  3. Lo detesta María.
- (3)
  1. She hates him.
  2. Lo detesta.

在英语中，无论宾语是完整名词（*John*）还是代词（*him*），都只能使用主语—动词—宾语（SVO）语序，如（1a）、（2a）与（3a）。无论句子是简单陈述句还是 *yes/no* 问句，这一点都成立。然而在西班牙语中，尽管 SVO 确实是原型语序，但也存在 SOV（如 2b 中代词的情形）、OVS（如 1c 与 2c 中完整名词与代词的情形）以及 OV（主语为空，即不表达，如 3b）。[8]在西班牙语中，OV 与 OVS 在 *yes/no* 问句里相当常见，在简单陈述句中也并不少见，并且在包含某些动词的句子中是原型语序。那么，当一个语言学习者——例如 L1 为英语的学习者——第一次遇到（并持续遇到）OVS 与 OV 类型句子时，会发生什么？研究表明，这类学习者会误解这类句子，并把「谁对谁做了什么」颠倒过来。在 *A Juan María lo detesta* 的情形中，学习者会把它理解为「John hates Mary」而不是「Mary hates John」。在 *Lo detesta María* 的情形中，他们会把句子理解为「He hates Mary」而不是「Mary hates him」。结果是形成了错误的形式—意义联结（例如 *lo* = he［subject］而非 *lo* = him［object］），并把关于句子结构的错误数据提供给负责存储与组织语言的内部加工器；在这个例子里，这些加工器收到的错误信息是：西班牙语具有严格的 SVO 语序，代词系统因此被搞得一团糟。

问题如下：这种解析问题是由于某种普遍策略 [9] 造成的，还是由于英语解析器与西班牙语输入数据互动造成的？在以往研究中，我采取的立场是：这是一种普遍策略，并提出了如下原则：

> **首名词原则。** 学习者倾向于把句子中遇到的第一个名词或代词加工为主语。

在这一普遍立场下，任何学习者，无论其来自 SVO 语言、语序灵活的语言，还是语序严格为 OVS 的语言，起初都会把第一个名词当作主语来加工。

在另一种立场下，如果 L1 解析器被迁移到 L2 的 IP 中，那么该原则会呈现不同形式，并带来不同后果。原则可能会是这样：

> **L1 迁移原则。** 学习者在开始习得时使用 L1 的解析程序。

在这种情况下，问题会因语言而异，取决于迁移。于是，说意大利语的人学习西班牙语并不会在西班牙语的 OV 与 OVS 结构上遇到困难，因为意大利语中也存在这些结构（例如 *Lo vede Maria*「Maria sees him」），而 L1 解析器拥有处理这些结构的计算机制。相反，英语母语者会遇到困难，因为英语语序严格，且没有处理非 SVO 结构的解析机制（裂句如「Him I hate」除外）（有关该问题的一项示例研究见 Isabelli，2008）。

围绕这个例子会出现一个问题：迁移究竟是解析机制本身的迁移，还是词汇迁移？西班牙语与意大利语共享如 *lo* 与 *la* 这样的宾格代词，因此意大利语母语者学习西班牙语时，可以把这些词汇项连同其功能特征一起迁移到新词库中；英语母语者则无法做到这一点。*lo* 的底层特征禁止它在意大利语中被当作主语，而对这些学习者而言，这种禁止推定也会在西班牙语作为 L2 时起作用。在这种情形下，意大利语 L1 学习者起初以首名词原则开始处理西班牙语 L2；但当该学习者访问带有从意大利语借来的特征的 *lo* 时，这一原则便与该词汇项的底层特征发生冲突。

另一方面，关于被动句习得的研究表明：像英语与法语这样「逐词对应」的被动结构等价并不会迁移，因此法语早期学习者仍倾向于在「谁对谁做了什么」上误解被动句（Ervin-Tripp，1974）。因此，L1 在基础 IP 中是否以及在多大程度上产生影响仍是一个开放问题；以及，这种影响究竟是实际的加工程序，还是词汇层面的影响。在另一项同时研究意大利语与英语（作为 L1 与 L2）的研究中，Gass（1989）指出：尽管各组在如何解释首名词上存在一些差异，但在不考虑有生性（animacy）时，各组呈现相同模式（关于会削弱首名词原则的原则见下文；另见 Tight，2012）。

其他因素也可能影响学习者的解析方式，从而影响他们对句子的解释。考虑动词 *scold*。在现实世界里，更可能发生的是父母责骂孩子，还是孩子责骂父母？第一种情形更常见。那么，如果学习者听到「The child scolded the mother」，会发生什么？在这种情况下，现实情景的概率可能（但不一定）会覆盖首名词原则（或其替代版本 L1 迁移原则）。学习者可能错误地重新解析句子，使其意味着「the parent scolded the child」，并把信息发送给内部加工器，让它们认为该语言存在 OVS 结构（即便事实上可能不存在）。在这种情形下，解析过程中会发生如下事情：the child = NP = subject，scolded = verb，the parent = NP = object；但等等，孩子不会责骂父母，父母才会责骂孩子，所以这句话一定是父母责骂孩子；重新分析解析结果：the child = NP = object，scolded = verb，the parent = NP = subject。所谓事件概率（event probabilities）的影响由如下原则捕捉：

> **事件概率原则。** 在可能的情况下，学习者可能依赖事件概率而非首名词原则来解释句子。

类似地，学习者在解析时也知道：某些动词要求某些情景成立。例如，动词 kick 要求行动者是具有腿的有生体。因此，人、马、青蛙甚至狗都能 kick，但蛇、岩石和病菌不能 kick。当面对句子「The cow was kicked by the horse」时，首名词原则（或 L1 迁移原则）可能导致误解：牛在 kick。可是，当面对句子「The fence was kicked by the horse」时，错误解释就不太可能（栅栏怎么 kick 任何东西？），这句话反而可能促使解析器重新分析它刚刚计算出的结果（前提是有时间这么做）。这种情形涉及所谓 *lexical semantics*。词汇语义（lexical semantics）指的是：动词意义如何对名词提出要求，使某个动作或事件得以发生。该动词表达的事件是否要求一个有生体来促成？是否要求某个实体具有特定属性？请注意，词汇语义与事件概率在根本上不同：在事件概率中，两种名词都可能执行该动作，只是其中一个更可能；而在词汇语义中，只有一个名词具备执行该动作的能力。因此，孩子与父母都可以 scold，只是其中一方更可能 scold 另一方（事件概率）；但在「horse」与「fence」之间，horse 可以 kick 别的东西，而 fence 不可能 kick 别的东西（词汇语义）。在解析中使用词汇语义可以由如下原则表达：

> **词汇语义原则。** 在可能的情况下，学习者可能依赖词汇语义而非首名词原则（或某种 L1 解析程序）来解释句子。

有关 L2 输入加工的研究还表明，语境会影响学习者如何解析句子。考虑如下两句话：

- (4a) John is in the hospital because Mary attacked him.
- (4b) John told his friends that Mary attacked him.

在西班牙语中，嵌套从句既可以是 SOV（*María lo atacó*），也可以是 OVS（*lo atacó María*）。如果首名词原则或其 L1 替代版本在起作用（例如对英语母语者而言），那么 OVS 结构可能会被误解为「he attacked Mary」。但请注意：如果前置语境是「John is in the hospital」，误解就不太可能。为什么 John 会在医院里，如果他袭击了 Mary？他更可能在监狱里。更合理的是：他因受伤而在医院里，因此一定是 Mary 袭击了他。如果前置语境是中性的，如「John told his friends …」，就没有任何东西约束后续从句的解释：John 完全可能告诉朋友他袭击了 Mary，也可能告诉朋友 Mary 袭击了他（见 VanPatten & Houston，1998）。因此，语境效应带来另一个原则：

> **语境约束原则。** 如果前置语境限制了某个从句或句子的可能解释，学习者可能较少依赖首名词原则（或 L1 迁移）。

到目前为止，我们讨论了在加工过程中影响形式与意义联结的因素，以及解析（例如对句法结构的计算）。还有另一个加工领域也会进入视野：元素更可能在句子中的哪些位置出现。想象你听到下面这一组数字：

> 11 32 51 4 8 42 71 39 7 22 60 15 96 12 85 44

如果你是典型情况，你会先记住开头的数字（例如 11、32），再记住结尾的数字（例如 44、85），而这两者又都会先于中间的任何数字（例如 39、7 或 60）。这种「最擅长加工并记住开头的内容，其次是结尾，最后是中间」的能力存在于相当大一部分人类信息加工中，语言也不例外。我们可以用如下原则来表述这一现象：

> **句子位置原则。** 学习者倾向于先加工位于句首的项目，再加工位于句末的项目，最后加工位于句中位置的项目。

Barcroft 与 VanPatten（1997）发现：在西班牙语中，语法语素 *se* 的初次觉察更常发生在像 *Se levanta Juan temprano todos los días* 这样的句子里，而不是 *Todos los días Juan se levanta temprano*「John gets up early every day」这样的句子里；前者中该语素更频繁地被未受提示的西班牙语初学者检测到。

当然，刚刚概述的这些原则（以及任何其他可能影响 IP 的原则 [10]）并不会孤立地起作用。例如，即便西班牙语中的宾语代词可以、也确实会出现在句首位置（例如在 OVS 结构中），这并不意味着学习者就会正确加工它们。首名词原则很可能会与宾语代词发生互动：学习者可能确实会加工该宾语代词，因为它在 OVS 结构中出现在句首（而不是像它通常那样出现在句中位置），但他们会把它加工错。这里需要牢记的是，「process」指的是学习者把意义与形式连接起来，无论是在局部层面（词、形态）还是在句子层面。正如我稍后将讨论的那样，processing 并不等同于「noticing」。

### 什么算作证据？

显然，只有在以理解为导向的研究中收集到的数据，才适合用来对 IP 作出推断。典型研究设计包括句子解释任务与眼动追踪实验。

#### 句子解释任务

在这种实验中，参与者听到句子并指出他们理解到的内容。例如，在语序问题上，参与者可能听到「The cow was kicked by the horse」。随后，他们会被要求在两张图片之间做选择，以代表他们听到的内容：一张图里牛在踢马；另一张图里马在踢牛。如果参与者选择第一张图，我们便可推断首名词原则正在引导句子加工；如果参与者选择第二张图，则首名词原则并未引导句子加工（见例如 VanPatten，1984）。

就形式—意义联结而言，这类任务可能会有一种变体。参与者可能听到诸如 *Mi mamá me llamó por teléfono*「My mother called me on the phone」与 *Mi amigo me ayuda con la casa*「My friend is helping me with the house」这样的句子。注意，这些句子里没有时间副词。同时，参与者也可能听到类似句子，但其中带有时间副词，例如 *Mi mamá me llamó anoche por teléfono*「My mother called me on the phone last night」。随后，学习者会被要求指出该动作是发生在过去、正在发生、每天发生，还是将来会发生。如果学习者在句子中不存在副词时无法正确作出判断，但在存在副词时却能正确判断，这就说明他们依赖词汇项来获取语义信息，而不是依赖动词屈折。

#### 眼动追踪

眼动追踪研究让参与者在电脑屏幕上阅读句子或文本，同时用微型摄像机通过照射到瞳孔的极小红外光追踪眼球瞳孔运动。当人们阅读时，他们会在逐毫秒层面无意识地跳过词与词的一部分、回视某些词等（关于 L2 眼动追踪研究的概述见 Keating，2013）。例如，眼动追踪可以揭示：学习者在 IP 过程中是否关注动词屈折；当遇到某些「不太对劲」的东西时，他们是否像母语者那样回视。例如，在句子「Last night my mother calls me on the phone」中，母语者的眼动追踪会显示：他们经常在动词 calls 上出现注视点，并回视到短语 last night。我们在初级与中级学习者身上看不到同样的眼动行为。然而，当被要求按按钮指示该动作属于过去、现在或将来时，母语者与非母语者总是按「过去」。这些结果结合起来表明：在加工句子时，学习者确实依赖词汇线索来获取意义，并会跳过那些编码同一意义的语法标记。

除眼动追踪之外，还有其他在线方法可用于研究 IP。读者可参阅 Jegerski 与 VanPatten（2013），该文集专门讨论 L2 研究中使用的心理语言学方法。

### 常见误解

关于 IP 以及本章所描述的具体 IP 模型，存在若干常见误解。

**误解 1：IP 是一个习得的模型/理论。** 持此观点的人认为 IP 试图更一般地解释习得。事实并非如此。如前所述，IP 只关心学习者如何建立形式—意义联结和/或如何解析句子。习得还涉及其他过程，包括对数据的容纳（数据如何被纳入发展中的语言系统，以及为什么可能不会被纳入）、普遍语法如何作用于数据、重组（被纳入的数据如何影响系统，例如当规则形式导致不规则形式被规则化时）、学习者如何产生输出、互动如何影响习得，等等。简而言之，**IP 只关心初始数据的收集。** 请考虑如下类比：酿蜜。蜜蜂要酿蜜；为此，它们必须采集花蜜。它们会去某些花而不去另一些。它们必须找到去花朵的路，并再回到蜂巢。随后它们加工花蜜以产出蜂蜜。它们建造巢房来储存蜂蜜，等等。这些努力都是酿蜜的一部分。但我们可以把研究聚焦于这样的问题：蜜蜂如何采集花蜜？它们为什么选择某些花而不选择另一些？这与 IP 的关切类似：学习者如何建立形式—意义联结？为什么在某个时期他们会建立某些联结而不会建立另一些？IP 把习得的一部分单独抽离出来；它关心的是习得的「采蜜环节」，并把「花蜜到了蜂巢之后发生什么」留给其他模型与理论去解释。（关于为何需要多种理论来解释习得这一复杂图景，见 Rothman & VanPatten，2013，以及 VanPatten，2014a，2018。）

**误解 2：输入加工否认输出、社会因素及其他因素的作用。** 在这种情形下，某人认为：因为研究聚焦于习得的某一方面，研究者或学者就认为其他任何因素都不重要。于是我们会听到诸如「输入 vs. 输出之争」或 SLA 中的「理解 vs. 产出」等说法。再次回到酿蜜类比：显然，研究蜜蜂如何采集花蜜、以及它们为何如此采集的人，会非常清楚采蜜并不等于酿蜜；同样，研究花蜜到达蜂巢后发生什么的人，也会清楚没有花蜜就没有酿蜜。研究者把注意力集中在习得拼图中的某一块，并不意味着他或她否认剩余部分；这只是意味着研究者在界定并深入检视拼图的一块。

**误解 3：输入加工等同于「noticing」。** 一些 IP 研究的读者错误地把 processing 等同于 noticing。提醒读者：processing 意味着学习者正在连接意义与形式。例子包括：/kaet/ 意味着「cat」；不过，对学习者而言，这一概念在心智中已有表征；/tahkt/ 意味着 talking 发生在过去时间框架中；而「John was told a lie by Mary」意味着 Mary 对 John 撒了谎。Schmidt（例如 Schmidt，1990，2001）对 noticing 的定义并不包含形式与意义的连接。noticing 仅仅意味着学习者意识到了语言的某个形式特征（包括新词）。此外，noticing 并未被应用到句子层面；它几乎总是局限在形态—词汇形式层面。两者的区分很重要，因为在一些出版物中，研究者曾反驳本章概述的原则；然而，他们的研究方法使用的是用于 noticing 而非 processing 的技术，包括知识测量（例如语法可接受性判断任务）、内省式口头报告（例如「告诉我你在阅读内容里注意到了什么」）以及标注任务（例如「阅读时，把任何吸引你注意的内容圈出来」）。不能用研究 noticing 的范式来反驳与 processing 相关的原则（见例如 VanPatten，2014b，2015）。

**误解 4：输入加工是一种以意义为基础的习得研究取向，因而忽视了我们对句法过程的认识。** 提出这种主张的人关注的是模型中的某些方面，例如词汇优先以及从输入中获取意义的追求如何驱动句子解释。他们的结论可以理解，但并不正确。正如我们在首名词原则与解析问题上看到的那样，该模型也关心解析的句法面向，以及这些面向如何影响句子解释与加工（从而影响习得）。此外，一些认为 IP 忽视句法过程的人，可能想到的是我们对成年母语者句子解释模型的认识：这些模型在很大程度上（但并非完全）以句法为主。有人会问：如果母语者加工模型如此，那么 L2 模型不也应当如此吗？答案也许是。我们在 IP 研究中采取的立场是：除了这里描述的这类原则之外，加工机制会随时间发展。学习者起步时所拥有的，可能并不是能够像母语者那样在句子解释中充分利用句法过程的加工机制。例如，在一项实验中，研究者表明母语者与非母语者对「空缺」（gap）句的加工不同。所谓空缺句，是指某个 *wh-* 成分（例如 *who*）从句子的一部分移位到另一部分，如：「The nurse who the doctor argued that the rude patient had angered is refusing to work late。」在这种句子中，*who*（关系从句标记）实际上与动词 *angered* 相连（即是动词 *angered* 的宾语）。研究者注意到：尽管母语者与非母语者都能同样判断谁对谁粗鲁，但逐毫秒的加工显示两者在利用句法加工方面存在显著差异：非母语者更依赖词汇语义与其他非句法线索。这被称为浅层结构假说（Shallow Structure Hypothesis，Clahsen & Felser，2006）。这并不意味着 IP 模型与 SSH 等价；这里的要点是：人们可以加工句子，但并不充分利用句法资源。

**误解 5：输入加工是一种教学取向。** 有些人认为本章描述的 IP 模型是一个教学模型。这是因为存在一种名为 *processing instruction* 的教学干预（有些人错误地称其为「input processing instruction」），它源自关于 IP 的洞见。加工教学关注的问题是：如果我们知道学习者在 IP 层面做错了什么，我们能否设计一种以理解为基础的教学干预，把他们从非最优加工推向更优加工？然而，IP 并不关心教学，也不关心课堂学习者在做什么。作为一个加工模型，它旨在适用于所有语境下、所有语言的所有学习者（课堂内外皆然）。因此，首名词原则既可以在美国移民学习英语的学习者身上研究，也可以在加拿大课堂学习英语的学习者身上研究，也可以在沙特阿拉伯课堂学习英语的学习者身上研究，等等。该模型试图描述学习者「自行」做什么，就像普遍语法研究试图描述学习者不受教学影响时「自行」做什么一样。话虽如此，加工教学的研究数量远多于 IP 本身的研究。也许正因如此，文献中出现了一些混淆。

### 代表性研究：Wong and Ito（2018）

在这项研究中，Wong 与 Ito（2018）试图检验一种受 IP 启发的特定教学干预——加工教学（processing instruction）——是否会改变以实时指标测量的加工行为。他们在两个实验中考察了大学第三学期的法语 L2 学习者（L1 为英语）。加工问题是首名词原则，以及该原则与包含动词 *faire*（「to make」）的法语使役结构的交互。英语与法语在构造使役句时略有不同，如（5）与（6）所示。

- (5) Henry makes Phillip clean the kitchen.

- (6) Henri fait nettoyer la cuisine à Phillipe.

  lit: Henry makes to clean the kitchen to Phillip.

不同之处在于：在动词「make」与 *faire* 之后的嵌套从句中，英语里 clean 的主语/施事位于动词之前；而法语里 *nettoyer* 的主语/施事位于动词之后，并由介词标记。首名词原则的预测是：法语学习者会把他们遇到的第一个名词（此处为 Henri）一路带到第二个动词 *nettoyer* 上，并「假定」他是该动词的施事/主语。简而言之，他们可能并不会真正加工动词 *fait*，从而把句子解释为类似「Henry cleans the kitchen for Phillip」。以往研究（VanPatten & Wong，2004）显示情况确实如此。

Wong 与 Ito 的做法是把参与者分成两个处理组：一组接受加工教学（源自 IP 原则的洞见）以及传统教学（包括对某一结构的呈现加练习，此处为使役结构）。参与者在处理前后都完成两项测评。一项用于测试基本句子理解，要求选择与所听内容匹配的正确图片（例如，要么是 Henry 站着而 Phillip 在打扫厨房，要么是 Phillip 在看着而 Henry 在打扫厨房）。另一项测评是使用所谓视觉世界范式（Visual World Paradigm）的在线眼动追踪实验。在这种实验中，参与者一边听句子，一边观看两张或更多图片。他们的无意识眼动（例如先看哪张图、是否以及何时转向另一张、是否回到另一张图等）会被记录。

在处理前测评中，Wong 与 Ito 发现参与者对法语使役结构的解释表现很差：他们选择正确图片的比例不到 10%。在眼动追踪中，他们的视线似乎从一开始就停留在错误图片上，往往不会看向正确图片。本章更有意义的是处理后的结果。在理解测试中，加工教学组的得分显著提升到约 62%，而传统组的得分仅略升至约 11%。在眼动追踪中，加工组在处理后开始呈现不同眼动：更频繁地看向正确图片，并且会回视到正确图片；传统组则没有这种变化。当研究者加入显性信息（他们的实验 2）后，结果基本相同，只是传统组的理解得分提升到 30%，其无意识眼动也略微向正确图片移动；加工组在加入显性信息后，其理解得分与眼动模式都没有变化。

该实验结果在多个方面都很有意思。第一，它显示显性信息（本章稍后会讨论）几乎无法被参与者用于改变类似 IP 这类行为或知识。第二，研究者对加工教学组提升幅度不更大感到有些意外，并据此认为首名词原则是一种相当强的策略，需要在输入中经历相当多次接触才能克服。（有关类似结果的有趣对比见 VanPatten，Borst，Collopy，Qualin，and Price（2013）的研究，该研究把首名词原则应用到四种不同语言：西班牙语、俄语、德语与法语。）简言之，研究者认为学习者使用的首名词策略可能相当顽固。

### 对 SLA 中观察到的发现的解释

**观察 1：接触输入是 L2 习得的必要条件；观察 2：大量 L2 习得是偶然发生的。** 不言而喻，IP 将输入的重要作用纳入其中。但更进一步，IP 模型会认为：大部分习得是偶然发生的。正如我们先前所指出的，IP 依赖理解（学习者积极投入于从所听或所读内容中获取意义）。在某种意义上，习得是学习者积极尝试理解输入的副产品。他们的主要注意力在意义上；形式—意义联结的建立以及句子的解析，都是学习者交际努力的结果。

**观察 4：学习者的输出（言语）在某一结构的习得中往往遵循可预测的路径，并呈现可预测的阶段。** 尽管 IP 的目标并不是解释 L2 习得的全部，但它可以帮助解释某些已观察到的现象。由于它关心的问题是「为什么学习者会建立某些形式—意义联结而不会建立另一些」，它可以讨论习得顺序。综合来看，IP 的各项原则可以解释为什么例如英语的动词屈折系统会以那样的方式出现。学习者会先加工（并随后习得）*-ing*，这既来自有意义先于无意义原则，也来自词汇偏好原则（没有任何词汇项承载 *-ing* 的意义）。第三人称 *-s* 会最后被习得，因为非冗余偏好原则（第三人称 *-s* 总是冗余的，而英语中其他动词屈折并非如此）。同样地，例如英语否定的初始习得阶段，通常以用特定词语来表达否定为特征：尤其是 no 与未经分析的 don’t（也就是说，学习者并不知道 don’t 由两个词 do 与 not 构成，而只是把它当作 no 的替代品使用）。这表明：初始 IP 正在试图分离内容词来表达否定。

**观察 7：频率对 L2 语言习得的影响存在上限；观察 8：学习者第一语言对 L2 习得的影响存在上限。** 在 IP 中，频率并不是主要因素。由于 IP 关注初始加工及其影响因素，频率不会发挥主要作用。例如，西班牙语中形容词一致非常常见，但冗余相关原则会抑制对一致的初始加工。其他一些不那么常见的东西，只要不冗余，反而会更早被加工。频率的问题在于：它有时与冗余/非冗余相伴出现。例如，在英语中，*-ing* 可能比简单过去时 *-ed* 更常见；但 *-ing* 也从不冗余，而 *-ed* 往往是冗余的（见前文）。因此问题变成：究竟是频率使 *-ing* 比 *-ed* 更早被加工，还是如词汇偏好原则所暗示的那样，是 *-ing* 的非冗余与意义性使其更早被加工？这类问题只能通过对多种语言的持续研究来回答。

IP 可以解释频率与 L1 影响存在上限。与词汇偏好、非冗余、有意义先于无意义等相关的各种原则，会削弱纯粹频率效应，也会削弱 L1 的影响。例如，仅仅因为某个形式出现得极其频繁，并不意味着它会被加工，如果（a）它是冗余的和/或（b）它不承载意义。同时，如果解析策略最终证明至少部分是普遍的，而非基于 L1（见首名词原则的讨论），那么 IP 模型就更能解释 L1 影响为何有限。

**观察 9：教学对 L2 习得的影响存在上限。** 当前的 IP 模型也有助于解释教学效果为何有限。相当多的教学以结果为中心，而不是以过程为中心。也就是说，教学往往关注规则与学习者输出。而我们的 IP 模型认为：学习问题的一部分在于加工。因此，如果教学没有解释输入中的内容如何被加工，它可能就没有我们想象的那么有用。围绕 IP 的研究产生了一种名为加工教学的教学干预，它正是针对这一问题。在加工教学中，教学试图在 IP 过程中进行干预，从而改变学习者的加工行为，并带来更「语法丰富」（甚至可以说更「恰当」）的 intake。

**观察 10：输出（学习者产出）对语言习得的影响存在上限。** 尽管 IP 并不直接讨论输出问题，但该模型会认为：如果输出不能帮助改变学习者的加工行为，那么学习者输出的效果会受到限制。例如，一个以英语为母语的西班牙语学习者可以在各种语境中产出无数句子；但如果互动并没有让学习者意识到自己误解了某个 OVS 句子，那么习得层面就不会有太大变化。该学习者仍会把西班牙语中的第一个（代）名词加工为主语。在这种情形下，输出之所以有用，是因为它能让学习者记录并修正自己对他人意义的误解（相关讨论见 VanPatten，2004c）。

### 显性/隐性之争

本章所呈现的 IP 模型在成人学习第二语言时是否动用隐性或显性过程这一问题上保持中立/不可知立场；不过，VanPatten（2015 以及其他著作）近来开始明确提出：显性加工不可能是 IP 的重要组成部分。在大多数解析与加工模型中，句法计算与映射发生在意识之外，或许学习词汇项是一个例外（见例如 Truscott & Sharwood Smith，2004，2011）。事实上，如果学习者在每个词或每一条形态数据上都停下来进行显式登记（例如「这是一个动词，它的意思是 X，它指涉第三人称，它是过去时」等），加工将会变得极其繁重；这种显性加工会让理解停摆。人们在听或读时确实会经历「what？」的瞬间：在极短的毫秒内意识到他们所听或所读的内容并不是所要表达的意思。但即便在这种情况下，我们也不清楚「所要表达的意思」的解决是否显性发生；也就是说，意识到有问题，并不必然意味着意识到如何解决问题（例如「哦，那是一个缩减关系从句而不是主句动词……」「哦，那是一个过去时动词形式而不是现在时动词形式……」）。（更多讨论见 VanPatten（2015）。）并且，有一些证据表明：就显性加工而言——也就是显性教授关于语言的信息，以及这些信息是否能在在线理解中被使用——显性信息通常不会、也很可能无法被用来加工语言（见例如 VanPatten et al.，2013）。

### 结语

作为一种现象，IP 应被视为我们称之为「习得」的复杂过程集合中的一部分。因此，任何 IP 的模型或理论都不应被期待成为更一般意义上的习得模型或理论。理想情况下，我们希望看到若干模型分别解释习得中的不同过程；当以这种方式来看待时，我们应当能得到一幅更好的习得图景。

模型与理论会发生改变与演化，L2 的 IP 模型也不例外。与 SLA 中几乎每一种理论与模型一样，IP 也遭遇过挑战，并在专业文献中引发了热烈争论（见例如 DeKeyser，Salaberry，Robinson，& Harrington，2002；VanPatten，2002 的一组交锋；以及 Harrington，2004；Carroll，2004；VanPatten，2004b 的另一组交锋）。然而，这些挑战针对的是模型的具体细节，而非驱动模型的底层问题，即：「为什么会建立某些形式—意义联结而不是另一些？在什么条件下？」某种 IP 模型必须与其他模型共存：后者处理语言数据如何被纳入发展中的系统，以及学习者如何访问该系统以产生输出等问题。当前的 IP 模型是我们对学习者在实时理解中如何加工输入这一问题的第一轮尝试。

### 讨论问题

1. IP 理论主张：在 IP 中词汇项具有优先地位（即内容词优先原则）。你认为在更一般意义上的习得中，词汇项也具有优先地位吗？学习者试图产出语言时又如何？学习者在显式尝试学习一门语言时（例如为了理解他人所说内容而采取的有意识策略）又如何？
2. VanPatten 认为，L2 解析可能涉及普遍程序，也可能在起初以 L1 为基础（即「迁移」了 L1 解析器），或者两者结合共同起作用。你认为哪种更可能？你能否设想额外的实验与数据来帮助判断哪种立场更可能？
3. 本章的 IP 理论主张：学习者面对输入时的初始取向是为意义加工，也就是说，他们会尽其所能从句子中提取基本意义。你能否想到任何情形，学习者会先为形式/结构加工句子？你认为这会导致习得吗？请牢记 processing 的定义是形式与意义的连接。
4. 以你所教授或最熟悉的语言为例，尝试把词汇偏好原则或首名词原则应用到该语言的 IP 中。你能否对加工问题作出一些预测？例如，在词汇偏好原则下，该语言的哪些形式特征倾向于与表达相同概念的词汇项或短语共现？你对加工的预测是什么？
5. IP 模型最广为人知的产出之一是 VanPatten 的加工教学（processing instruction）。请选择下列研究之一并在课堂上介绍：Cadierno（1995）、VanPatten et al.（2013）、Uludag and VanPatten（2012）。
6. 阅读本章的代表性研究，并准备一份课堂讨论：描述你将如何开展一项复现研究。务必说明你会做出哪些改变，以及这些改变的动机是什么。
